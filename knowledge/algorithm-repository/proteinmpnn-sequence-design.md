**Source**: Dauparas, J., Anishchenko, I., Bennett, N., Bai, H., Ragotte, R. J., Milles, L. F., Wicky, B. I. M., Courbet, A., de Haas, R. J., Bethel, N., Leung, P. J. Y., Huddy, T. F., Pellock, S., Tischer, D., Chan, F., Koepnick, B., Nguyen, H., Kang, A., Sankaran, B., ... Baker, D. (2022). Robust deep learning-based protein sequence design using ProteinMPNN. *Science*, 378(6615), 49--56. https://doi.org/10.1126/science.add2187

**Category**: Bioinformatics / Protein Design and Structural Biology

## Biological / Computational Problem

The **inverse protein folding problem** asks: given a target protein backbone 3D structure, what amino acid sequence will fold into that structure? This is fundamental for protein engineering -- designing new proteins with therapeutic, catalytic, or material applications. Previous physics-based methods (Rosetta) were slow and required expert customization.

- **Input data**: Protein backbone 3D coordinates (N, C-alpha, C, O atoms for each residue)
- **Output**: Predicted amino acid sequence that stabilizes the given backbone structure

## Mathematical / Computational Model

### Message-Passing Neural Network on Protein Graphs

ProteinMPNN represents the protein backbone as a **k-nearest neighbor graph** where:
- Nodes = amino acid residues (backbone atoms: N, C-alpha, C)
- Edges = spatial proximity (based on C-beta distances)

### Encoder-Decoder Architecture

The model uses an **encoder-decoder architecture** with message passing:

**Encoder**: Encodes backbone geometry into per-residue and pairwise features.

For each residue $i$, the encoder computes:
- **Node features**: dihedral angles ($\phi, \psi, \omega$), backbone angles ($\theta, \tau$)
- **Edge features**: distance between C-beta atoms $d_{ij}$, relative orientation features

The encoder message-passing updates:

$$\mathbf{h}_i^{(t+1)} = \mathbf{h}_i^{(t)} + \text{ReLU}\left(\text{BN}\left(\sum_{j \in \mathcal{N}(i)} \mathbf{W}_{\text{enc}} \left[\mathbf{h}_i^{(t)} \| \mathbf{h}_j^{(t)} \| \mathbf{e}_{ij}\right]\right)\right)$$

where $\mathcal{N}(i)$ are the $k$ nearest neighbors of residue $i$, $\mathbf{e}_{ij}$ are edge features, and $\|$ denotes concatenation.

**Decoder**: An autoregressive decoder that predicts amino acid identities sequentially.

For each position in a random order $\sigma$, the decoder predicts:

$$p(\text{aa}_i \mid \text{backbone}, \text{aa}_{\sigma(1)}, \dots, \text{aa}_{\sigma(i-1)}) = \text{softmax}(\mathbf{W}_{\text{out}} \mathbf{h}_i^{(D)})$$

where $\mathbf{h}_i^{(D)}$ is the decoder hidden state at position $i$, updated via:

$$\mathbf{h}_i^{(D)} = \text{ReLU}\left(\text{BN}\left(\sum_{j \in \mathcal{N}(i)} \mathbf{W}_{\text{dec}} \left[\mathbf{h}_i^{(E)} \| \mathbf{h}_j^{(E)} \| \mathbf{e}_{ij} \| \mathbf{a}_{\sigma(j)}\right]\right)\right)$$

where $\mathbf{h}_i^{(E)}$ is the encoder output for residue $i$, and $\mathbf{a}_{\sigma(j)}$ is the embedding of the amino acid at position $\sigma(j)$.

### Noise Injection for Robustness

A key innovation is **noise injection during training** -- backbone coordinates are perturbed with Gaussian noise:

$$\mathbf{x}_i^{\text{(noisy)}} = \mathbf{x}_i + \epsilon_i, \quad \epsilon_i \sim \mathcal{N}(0, \sigma^2 \mathbf{I})$$

with $\sigma$ ranging from 0 to 0.5 A during training. This makes the model robust to small structural errors, unlike Rosetta which fails with even minor inaccuracies.

### Loss Function

The model minimizes the **cross-entropy loss**:

$$\mathcal{L} = -\frac{1}{N} \sum_{i=1}^{N} \sum_{a=1}^{20} \mathbb{1}[\text{aa}_i = a] \log p(\text{aa}_i = a \mid \text{backbone}, \text{aa}_{<i})$$

## Key Assumptions

| Assumption | Formalization | Implication |
|-----------|--------------|-------------|
| Backbone is fixed | Input coordinates are known and static | Cannot design flexible/multi-state backbones |
| K-nearest neighbors suffices | Graph edges $\propto$ $d_{ij}^{-1}$ in top-$k$ | Very long-range interactions may be missed |
| Autoregressive decoding order is arbitrary | Any permutation $\sigma$ works | In practice, random order works well; no causal ordering needed in protein sequences |
| 20 amino acid types | Output space $\in \{1, \dots, 20\}$ | Does not account for non-canonical amino acids or post-translational modifications |

## Applicable Scenarios

**When to use**:
- Designing sequences for novel protein backbones (generated by RFdiffusion or hallucination)
- Rescuing failed designs from physics-based methods (Rosetta)
- Designing functional proteins (binders, enzymes, nanomaterials)
- Multi-chain protein complex design (oligomers, nanoparticles)

**When NOT to use**:
- Designing disordered proteins (no fixed backbone available)
- Predicting structure from sequence (use AlphaFold or RoseTTAFold instead)
- Systems requiring non-canonical amino acids
- Designing proteins with significant backbone dynamics

**Comparison**: ProteinMPNN achieves 52.4% sequence recovery vs. 32.9% for Rosetta, is $>$200x faster ($\sim$1 second vs. minutes for Rosetta), and works out of the box without expert parameter tuning.

## Implementation Details

- **Key parameters**: 3 encoder layers, 3 decoder layers, 128 hidden dim, 48 nearest neighbors
- **Computational requirements**: Inference on GPU: $<$1 second for a 500-residue protein; CPU: 5--10 seconds; training on PDB requires 1--4 GPUs for 2 days
- **Preprocessing**:
  - Parse PDB files to extract backbone atom coordinates (N, CA, C, O)
  - Standardize coordinates to remove crystal packing artifacts
  - Compute inter-residue distances and orientations
  - Filter by resolution ($<$3.0 A), sequence length ($<$1,000 residues)

"""
Minimal implementation of ProteinMPNN-style protein sequence design.

This provides a simplified message-passing neural network for predicting 
amino acid sequences from backbone coordinates, demonstrated on synthetic data.
"""

import math
import numpy as np
import torch
import torch.nn as nn
import torch.nn.functional as F

# For reproducibility
torch.manual_seed(42)
np.random.seed(42)


# Amino acid vocabulary
AA_ALPHABET = "ACDEFGHIKLMNPQRSTVWY"
AA_TO_IDX = {aa: i for i, aa in enumerate(AA_ALPHABET)}
IDX_TO_AA = {i: aa for i, aa in enumerate(AA_ALPHABET)}


def simulate_protein_backbone(
    n_residues: int = 100,
    seed: int = 42,
):
    """
    Generate a synthetic protein backbone (C-alpha trace) using a random walk
    with local geometry constraints (simulating Ramachandran-like angles).
    
    Parameters
    ----------
    n_residues : int
        Number of residues.
    
    Returns
    -------
    coords : Tensor (n_residues x 3)
        C-alpha coordinates (simulated).
    phi_psi : Tensor (n_residues x 2)
        Simulated backbone dihedral angles.
    true_sequence : str
        A random amino acid sequence for initialization.
    """
    rng = np.random.default_rng(seed)
    
    # Start at origin
    coords = np.zeros((n_residues, 3))
    
    # Bond length ~3.8 A (C-alpha to C-alpha)
    bond_length = 3.8
    
    # Generate random phi/psi angles (biased toward allowed regions)
    phi = rng.uniform(-180, 180, size=n_residues)
    psi = rng.uniform(-180, 180, size=n_residues)
    
    for i in range(1, n_residues):
        # Convert local angles to global coordinates
        if i == 1:
            # Second residue along x-axis
            coords[i] = coords[i-1] + np.array([bond_length, 0, 0])
        elif i == 2:
            # Third residue in the xy plane
            theta = np.radians(psi[i-1])
            coords[i] = coords[i-1] + np.array([
                bond_length * np.cos(theta),
                bond_length * np.sin(theta),
                0
            ])
        else:
            # Use phi, psi to rotate and extend
            phi_rad = np.radians(phi[i-1])
            psi_rad = np.radians(psi[i-1])
            
            # Direction from previous steps
            v1 = coords[i-1] - coords[i-2]
            v2 = coords[i-2] - coords[i-3]
            
            # Normal and binormal
            n_vec = np.cross(v1, v2)
            n_norm = np.linalg.norm(n_vec)
            if n_norm > 1e-8:
                n_vec = n_vec / n_norm
            else:
                n_vec = np.array([0, 0, 1])
            
            b_vec = np.cross(n_vec, v1)
            b_norm = np.linalg.norm(b_vec)
            if b_norm > 1e-8:
                b_vec = b_vec / b_norm
            else:
                b_vec = np.array([0, 1, 0])
            
            # Rotate by psi around the normal, then by phi around the bond
            v1_norm = v1 / np.linalg.norm(v1)
            direction = (np.cos(psi_rad) * v1_norm + 
                         np.sin(psi_rad) * np.cos(phi_rad) * b_vec +
                         np.sin(psi_rad) * np.sin(phi_rad) * n_vec)
            direction = direction / np.linalg.norm(direction)
            
            coords[i] = coords[i-1] + direction * bond_length
    
    # Generate a random "true" sequence (used only for geometry perturbation)
    true_seq = "".join(rng.choice(list(AA_ALPHABET), size=n_residues))
    
    return (
        torch.tensor(coords, dtype=torch.float32),
        torch.tensor(np.column_stack([phi, psi]), dtype=torch.float32),
        true_seq,
    )


def compute_distance_and_orientation_features(
    coords: torch.Tensor,
    k_neighbors: int = 30,
):
    """
    Compute pairwise distance and orientation features from backbone coordinates.
    
    Parameters
    ----------
    coords : Tensor (N, 3)
        C-alpha coordinates.
    k_neighbors : int
        Number of nearest neighbors.
    
    Returns
    -------
    edge_idx : Tensor (N, k_neighbors)
        Indices of k-nearest neighbors.
    edge_features : Tensor (N, k_neighbors, n_features)
        Edge features (distances, relative orientations).
    """
    N = coords.shape[0]
    
    # Pairwise distance matrix
    dist = torch.cdist(coords, coords)  # (N, N)
    
    # Mask self-distances
    dist.fill_diagonal_(float('inf'))
    
    # Get k-nearest neighbors
    dist_sorted, idx_sorted = torch.sort(dist, dim=-1)
    edge_idx = idx_sorted[:, :k_neighbors]  # (N, k)
    
    # Edge features
    # 1. Distance encoding
    d = dist_sorted[:, :k_neighbors]  # (N, k)
    d_feat = torch.exp(-d / 3.0)  # RBF-like encoding
    
    # 2. Relative orientation
    # Vector from residue i to its neighbor j
    pos_i = coords.unsqueeze(1).expand(-1, k_neighbors, -1)  # (N, k, 3)
    pos_j = coords[edge_idx]  # (N, k, 3)
    rel_vec = pos_j - pos_i  # (N, k, 3)
    rel_norm = rel_vec / (d.unsqueeze(-1) + 1e-8)  # (N, k, 3)
    
    # 3. Distance bins
    dist_bins = torch.linspace(3.0, 20.0, 16, device=coords.device)
    rbf = torch.exp(-((d.unsqueeze(-1) - dist_bins) ** 2) / 0.5)  # (N, k, 16)
    
    # Concatenate features
    edge_features = torch.cat([
        d_feat.unsqueeze(-1),  # (N, k, 1)
        rel_norm,              # (N, k, 3)
        rbf,                   # (N, k, 16)
    ], dim=-1)  # (N, k, 20)
    
    return edge_idx, edge_features


class ProteinMPNNEncoder(nn.Module):
    """
    Encoder for ProteinMPNN: message-passing on residue graph.
    """
    
    def __init__(self, d_node: int = 64, d_edge: int = 20, d_hidden: int = 128, 
                 n_layers: int = 3):
        super().__init__()
        self.d_node = d_node
        self.d_edge = d_edge
        
        # Node embedding
        self.node_proj = nn.Linear(3, d_node)  # from coordinates
        
        # Edge embedding
        self.edge_proj = nn.Linear(d_edge, d_hidden)
        
        # Message-passing layers
        self.mp_layers = nn.ModuleList()
        for _ in range(n_layers):
            layer = nn.Sequential(
                nn.Linear(d_node * 2 + d_hidden, d_hidden),
                nn.ReLU(),
                nn.Batchorm1d(d_hidden),
                nn.Linear(d_hidden, d_node),
                nn.ReLU(),
                nn.Batchorm1d(d_node),
            )
            self.mp_layers.append(layer)
    
    def forward(self, coords: torch.Tensor, edge_idx: torch.Tensor,
                edge_features: torch.Tensor) -> torch.Tensor:
        """
        Parameters
        ----------
        coords : Tensor (N, 3)
        edge_idx : Tensor (N, k)
        edge_features : Tensor (N, k, d_edge)
        
        Returns
        -------
        Tensor (N, d_node)
            Encoded node features.
        """
        N = coords.shape[0]
        
        # Node features from coordinates (local geometry)
        # Compute local coordinate frames
        h_node = self.node_proj(coords)  # (N, d_node)
        
        # Edge features
        h_edge = self.edge_proj(edge_features)  # (N, k, d_hidden)
        
        # Message passing
        for layer in self.mp_layers:
            # Gather neighbor features
            neighbor_feats = h_node[edge_idx]  # (N, k, d_node)
            
            # Node i feature broadcast to k neighbors
            node_feats = h_node.unsqueeze(1).expand(-1, edge_idx.shape[1], -1)
            
            # Message: concatenate [node_i, node_j, edge_ij]
            messages = torch.cat([node_feats, neighbor_feats, h_edge], dim=-1)
            
            # MLP on messages
            B, K, D = messages.shape
            messages_flat = messages.view(-1, messages.shape[-1])
            updates = layer[0](messages_flat)  # first Linear
            updates = layer[1](updates)
            # Batchorm1d: need shape (N*K, D)
            updates = layer[2](updates)
            updates = layer[3](updates)
            updates = layer[4](updates)
            updates = layer[5](updates)
            updates = updates.view(N, K, -1)
            
            # Aggregate by summation
            agg = updates.mean(dim=1)  # (N, d_node)
            
            # Residual connection
            h_node = h_node + agg
        
        return h_node


class ProteinMPNNDecoder(nn.Module):
    """
    Autoregressive decoder for ProteinMPNN.
    
    Predicts amino acid identities sequentially in random order.
    """
    
    def __init__(self, d_node: int = 64, d_edge: int = 20, d_hidden: int = 128,
                 n_layers: int = 3):
        super().__init__()
        
        # Amino acid embedding (20 types + 1 for unknown)
        self.aa_embedding = nn.Embedding(21, d_node)
        
        # Decoder message-passing layers
        self.mp_layers = nn.ModuleList()
        for _ in range(n_layers):
            layer = nn.Sequential(
                nn.Linear(d_node * 2 + d_hidden + d_node, d_hidden),
                nn.ReLU(),
                nn.Linear(d_hidden, d_node),
                nn.ReLU(),
            )
            self.mp_layers.append(layer)
        
        # Final output projection
        self.output_proj = nn.Linear(d_node, 20)  # 20 amino acids
    
    def forward(
        self,
        h_enc: torch.Tensor,
        edge_idx: torch.Tensor,
        edge_features: torch.Tensor,
        input_aa: torch.Tensor = None,
        temperature: float = 1.0,
    ) -> torch.Tensor:
        """
        Parameters
        ----------
        h_enc : Tensor (N, d_node)
            Encoder output.
        edge_idx : Tensor (N, k)
        edge_features : Tensor (N, k, d_edge)
        input_aa : Tensor (N,) or None
            Input amino acids for teacher forcing.
        temperature : float
        
        Returns
        -------
        logits : Tensor (N, 20)
        """
        N = h_enc.shape[0]
        
        # Random decoding order
        order = torch.randperm(N, device=h_enc.device)
        
        # Initialize with amino acid tokens
        if input_aa is not None:
            aa_tokens = input_aa
        else:
            aa_tokens = torch.full((N,), 20, dtype=torch.long, device=h_enc.device)
        
        aa_emb = self.aa_embedding(aa_tokens)  # (N, d_node)
        
        # Autoregressive decoding in random order
        h_dec = h_enc.clone()
        
        for pos in order:
            # Gather neighbor features
            neighbor_feats = h_dec[edge_idx[pos]]  # (k, d_node)
            
            # Edge features for this node
            edge_feats = edge_features[pos]  # (k, d_edge)
            edge_emb = self.edge_proj(edge_feats)
            
            # Broadcast current node features
            node_feat = h_dec[pos].unsqueeze(0).expand(neighbor_feats.shape[0], -1)
            aa_feat = aa_emb[pos].unsqueeze(0).expand(neighbor_feats.shape[0], -1)
            
            # Message
            msg = torch.cat([node_feat, neighbor_feats, edge_emb, aa_feat], dim=-1)
            
            for layer in self.mp_layers:
                msg = layer(msg)
            
            # Update
            agg = msg.mean(dim=0)  # (d_node,)
            h_dec[pos] = h_enc[pos] + agg
        
        # Final logits
        logits = self.output_proj(h_dec) / temperature
        
        return logits


class SimplifiedProteinMPNN(nn.Module):
    """
    Simplified ProteinMPNN for protein sequence design.
    
    Combines encoder and decoder into a unified model.
    """
    
    def __init__(
        self,
        k_neighbors: int = 30,
        d_node: int = 64,
        d_hidden: int = 128,
        n_encoder_layers: int = 3,
        n_decoder_layers: int = 3,
    ):
        super().__init__()
        self.k_neighbors = k_neighbors
        
        self.encoder = ProteinMPNNEncoder(d_node=d_node, d_hidden=d_hidden,
                                           n_layers=n_encoder_layers)
        self.decoder = ProteinMPNNDecoder(d_node=d_node, d_hidden=d_hidden,
                                           n_layers=n_decoder_layers)
        
        # Edge projection (shared)
        self.edge_proj = nn.Linear(20, d_hidden)
        self.decoder.edge_proj = self.edge_proj  # Share edge projection
    
    def forward(
        self,
        coords: torch.Tensor,
        input_aa: torch.Tensor = None,
        temperature: float = 1.0,
    ) -> torch.Tensor:
        """
        Predict amino acid sequence from backbone coordinates.
        
        Parameters
        ----------
        coords : Tensor (N, 3)
            C-alpha coordinates.
        input_aa : Tensor (N,) or None
            Ground truth sequence for teacher forcing.
        temperature : float
            Sampling temperature.
        
        Returns
        -------
        Tensor (N, 20)
            Logits for each amino acid.
        """
        edge_idx, edge_features = compute_distance_and_orientation_features(
            coords, self.k_neighbors
        )
        
        h_enc = self.encoder(coords, edge_idx, edge_features)
        logits = self.decoder(h_enc, edge_idx, edge_features, input_aa, temperature)
        
        return logits
    
    def design(self, coords: torch.Tensor, n_samples: int = 1,
               temperature: float = 1.0) -> list:
        """
        Design amino acid sequence(s) for a given backbone.
        
        Parameters
        ----------
        coords : Tensor (N, 3)
        n_samples : int
        temperature : float
        
        Returns
        -------
        list of str
        """
        sequences = []
        for _ in range(n_samples):
            logits = self.forward(coords, temperature=temperature)
            probs = F.softmax(logits / temperature, dim=-1)
            pred_tokens = torch.multinomial(probs, num_samples=1).squeeze(-1)
            sequence = "".join(IDX_TO_AA[t.item()] for t in pred_tokens)
            sequences.append(sequence)
        
        return sequences
    
    def compute_recovery_rate(
        self, coords: torch.Tensor, true_sequence: str
    ) -> float:
        """
        Compute sequence recovery rate (native sequence identity).
        
        Parameters
        ----------
        coords : Tensor (N, 3)
        true_sequence : str
        
        Returns
        -------
        float (0 to 1)
        """
        tokens = torch.tensor(
            [AA_TO_IDX.get(aa, 20) for aa in true_sequence],
            dtype=torch.long
        )
        logits = self.forward(coords, input_aa=tokens)
        pred_tokens = logits.argmax(dim=-1)
        
        match = (pred_tokens == tokens).float().mean().item()
        return match


class SequenceDesignTrainer:
    """
    Trainer for ProteinMPNN with noise injection.
    """
    
    def __init__(self, model: SimplifiedProteinMPNN, lr: float = 1e-3):
        self.model = model
        self.optimizer = torch.optim.Adam(model.parameters(), lr=lr)
        self.loss_fn = nn.CrossEntropyLoss()
    
    def train_step(
        self, coords: torch.Tensor, sequence: str, noise_std: float = 0.1
    ) -> float:
        """
        Training step with noise injection for robustness.
        """
        self.model.train()
        self.optimizer.zero_grad()
        
        # Add noise to coordinates (key ProteinMPNN innovation)
        noise = torch.randn_like(coords) * noise_std
        noisy_coords = coords + noise
        
        # Teacher forcing: provide true amino acids as input
        tokens = torch.tensor(
            [AA_TO_IDX.get(aa, 20) for aa in sequence],
            dtype=torch.long
        )
        
        logits = self.model(noisy_coords, input_aa=tokens)
        loss = self.loss_fn(logits, tokens)
        
        loss.backward()
        torch.nn.utils.clip_grad_norm_(self.model.parameters(), 1.0)
        self.optimizer.step()
        
        return loss.item()

def main():
    """
    Run a complete ProteinMPNN-style protein sequence design demonstration.
    """
    print("=" * 60)
    print("ProteinMPNN: Deep Learning Protein Sequence Design")
    print("=" * 60)
    
    # --- 1. Generate synthetic protein backbones ---
    print("\n[1] Generating synthetic protein backbones...")
    train_coords, _, train_seq = simulate_protein_backbone(
        n_residues=60, seed=42
    )
    test_coords, _, test_seq = simulate_protein_backbone(
        n_residues=60, seed=99
    )
    print(f"    Training backbone: {train_coords.shape[0]} residues")
    print(f"    Test backbone: {test_coords.shape[0]} residues")
    print(f"    Training sequence: {train_seq[:30]}...")
    
    # --- 2. Create model ---
    print("\n[2] Initializing ProteinMPNN model...")
    model = SimplifiedProteinMPNN(
        k_neighbors=20,
        d_node=32,
        d_hidden=64,
        n_encoder_layers=2,
        n_decoder_layers=2,
    )
    n_params = sum(p.numel() for p in model.parameters())
    print(f"    Model parameters: {n_params:,}")
    
    # --- 3. Train with noise injection ---
    print("\n[3] Training with noise injection (key innovation)...")
    trainer = SequenceDesignTrainer(model, lr=5e-3)
    
    n_epochs = 100
    for epoch in range(n_epochs):
        # Vary noise level during training
        noise_std = 0.05 if epoch < 30 else (0.1 if epoch < 60 else 0.2)
        loss = trainer.train_step(train_coords, train_seq, noise_std=noise_std)
        
        if (epoch + 1) % 20 == 0:
            print(f"    Epoch {epoch+1}/{n_epochs}, loss: {loss:.4f} "
                  f"(noise_std={noise_std})")
    
    # --- 4. Evaluate recovery rate ---
    print("\n[4] Sequence recovery on test backbone...")
    recovery = model.compute_recovery_rate(test_coords, test_seq)
    print(f"    Sequence recovery rate: {recovery:.3f} " +
          f"(expected: <native sequence identity>")
    
    # --- 5. Design new sequences ---
    print("\n[5] De novo sequence design...")
    designed = model.design(test_coords, n_samples=3, temperature=0.5)
    
    for i, seq in enumerate(designed):
        identity = sum(1 for a, b in zip(seq, test_seq) if a == b) / len(test_seq)
        print(f"    Design {i+1}: {seq[:40]}...")
        print(f"               Identity vs native: {identity:.3f}")
    
    # --- 6. Robustness test: varying noise levels ---
    print("\n[6] Robustness to structural noise...")
    for noise_std in [0.0, 0.1, 0.3, 0.5]:
        noisy_coords = test_coords + torch.randn_like(test_coords) * noise_std
        recovery_noisy = model.compute_recovery_rate(noisy_coords, test_seq)
        print(f"    Noise std={noise_std:.1f}: recovery rate={recovery_noisy:.3f}")
    
    # --- 7. Temperature exploration ---
    print("\n[7] Effect of sampling temperature on diversity...")
    for temp in [0.1, 0.5, 1.0, 2.0]:
        designs = model.design(test_coords, n_samples=5, temperature=temp)
        identities = []
        for seq in designs:
            ident = sum(1 for a, b in zip(seq, test_seq) if a == b) / len(test_seq)
            identities.append(ident)
        print(f"    T={temp:.1f}: mean recovery={np.mean(identities):.3f}, "
              f"diversity={np.std(identities):.3f}")
    
    print("\n" + "=" * 60)
    print("ProteinMPNN demo complete.")
    print("=" * 60)


if __name__ == "__main__":
    main()
