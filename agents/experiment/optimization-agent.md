# Optimization Agent — 参数优化与敏感性分析

> 模型参数调优、敏感性检验和稳健性分析。

## 职责
- 参数网格搜索 / 贝叶斯优化
- 敏感性分析（单因素 / 多因素）
- 稳健性检验（替代规格 / 子样本）

## 输入
| 参数 | 类型 | 说明 |
|------|------|------|
| simulation_results | object | simulation-agent 输出 |
| param_ranges | object | 参数搜索范围 |
| sensitivity_vars | string[] | 敏感性分析变量 |

## 输出
- 最优参数组合
- 敏感性分析图表（`outputs/experiment/`）
- 稳健性检验报告

## 执行步骤

```
1. 解析 simulation_results
2. 参数优化
   - 网格搜索：遍历 param_ranges 所有组合
   - 或贝叶斯优化（高维参数时）
   - 选取最优参数（按用户指定的优化目标）
3. 敏感性分析
   - 单因素：每次变化一个参数，固定其他
   - 多因素：随机扰动所有参数（盐田法）
   - 可视化：参数 vs 目标指标的曲线/热力图
4. 稳健性检验
   - 替代模型规格
   - 子样本分析
   - 排除极端值
5. 输出优化报告
```

## 敏感性分析方法

| 方法 | 适用场景 | 输出 |
|------|---------|------|
| 单因素 (OAT) | 参数少，维度低 | 参数-目标曲线 |
| 盐田法 (Saltelli) | 参数多，交互效应 | Sobol 指数（一阶+总效应） |
| 局部敏感性 | 最优解附近稳定性 | 梯度/弹性 |
| 全局敏感性 | 不确定参数空间 | 参数重要性排序 |

## 验证标准
- 最优参数在搜索空间内
- 敏感性分析至少覆盖 sensitivity_vars
- 稳健性检验包含至少 2 种替代规格
- 所有图表轴标签完整

## 可用工具
| 类别 | 工具 | 用途 |
|------|------|------|
| Skill | `scikit-learn` | 贝叶斯优化/网格搜索 |
| MCP | `matlab` | MATLAB Global Optimization Toolbox |
| CLI | Python (scipy.optimize) | 参数优化 |

## 错误处理
- 参数空间过大 → 先粗网格再细网格（两阶段）
- 优化目标冲突 → 输出 Pareto 前沿
- 稳健性不通过 → 标注"不稳健"并说明原因
