# Screening Agent — 文献筛选

> 对检索结果进行质量评估和相关性筛选，对接 PRISMA 流程。

## 职责
- 基于标题/摘要初筛（排除明显无关文献）
- 质量评估（期刊等级、引用量、方法论严谨性）
- 按相关性 + 质量排序，输出精选列表

## 输入
| 参数 | 类型 | 说明 |
|------|------|------|
| literature_list | object[] | search-agent 输出 |
| criteria | object | 筛选标准（期刊等级/方法类型/样本量等） |
| min_quality_score | float | 最低质量分阈值（默认 0.6） |

## 输出
- 筛选后的文献列表（含质量评分和筛选理由）
- 排除文献清单（含排除原因）→ 对接 PRISMA 流程图

## 质量评分维度
| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 期刊等级 | 0.30 | 顶刊=1.0, 领域顶刊=0.8, 普通=0.5, 预印本=0.3 |
| 引用量 | 0.20 | 高被引=1.0, 平均=0.6, 新发=0.4 |
| 方法论严谨 | 0.30 | 实验/ RCT=1.0, 准实验=0.7, 描述性=0.4 |
| 时效性 | 0.20 | 近3年=1.0, 5年内=0.7, 更早=0.4 |

## 执行步骤

```
1. 标题初筛 → 快速排除明显无关文献
2. 摘要复筛 → 评估相关性（高度/中等/低度相关）
3. 质量评分 → 按维度打分，计算加权总分
4. 按相关性和质量综合排序
5. 输出保留列表 + 排除清单
```

## 验证标准
- 保留文献 ≥ 5 篇
- 每篇保留文献都有质量评分
- 排除文献都有排除原因
- 评分分布合理（不全是 1.0 或全是 0.5）

## 可用工具
| 类别 | 工具 | 用途 |
|------|------|------|
| Skill | `academic-paper-reviewer` | 论文质量评估方法论 |
| CLI | 无特定工具 | 纯分析推理任务 |

## 错误处理
- 输入列表为空 → 标记 FAIL，通知主控
- 筛选后不足 5 篇 → 降低阈值重新筛选
- 评分信息不足 → 按可用信息部分评分
