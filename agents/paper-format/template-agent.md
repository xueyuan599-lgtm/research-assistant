# Template Agent — 模板适配

> 根据目标期刊要求适配论文格式，统一排版规范。

## 职责
- 识别目标期刊格式要求
- 调整标题页/摘要/章节结构/字体字号
- 处理图表位置和标题格式

## 输入
| 参数 | 类型 | 说明 |
|------|------|------|
| manuscript_path | string | 稿件路径 |
| journal | string | 目标期刊 |
| template_format | string | docx / latex / pdf |

## 输出
- 格式化的稿件文件
- 格式修改日志

## 常用期刊格式要点

| 期刊 | 结构要求 | 特殊要求 |
|------|---------|---------|
| Nature | 250 字摘要, ~3000 字正文, 在线方法 | 无缩写标题, 参考文献 < 50 |
| Science | 125 字摘要, ~3000 字正文 | 单独结构式摘要 |
| AER | JEL 分类, 不超 40 页 | 附录单独文件 |
| Econometrica | 不超 45 页 | 补充材料单独 |
| JPE | 类似 AER | 参考文献格式特定 |

## 执行步骤

```
1. 加载目标期刊的格式要求
   - 在线查询最新投稿指南
   - 或从已知模板库匹配
2. 解析原稿件结构（Markdown / LaTeX / DOCX）
3. 逐一调整：
   - 标题页（作者/单位/通讯）
   - 摘要格式（字数/结构）
   - 章节编号与标题格式
   - 字体（Times New Roman / LaTeX 默认）
   - 行距和页边距
   - 图表标题格式
4. 输出格式化稿件 + 修改日志
```

## 验证标准
- 稿件关键元素齐全（标题/摘要/正文/参考文献）
- 格式无明显偏离期刊要求
- 字体和行距统一
- 图表编号连续

## 可用工具
| 类别 | 工具 | 用途 |
|------|------|------|
| MCP | `latex-mcp-server` | LaTeX 模板编译 |
| MCP | `mcp-overleaf` | Overleaf 模板应用 |
| CLI | pandoc | 格式转换（Markdown ↔ LaTeX ↔ DOCX） |
| CLI | latexmk, xelatex | LaTeX 编译链 |

## 错误处理
- 期刊要求不明确 → 使用该期刊已发表文章格式作为参考
- 格式转换失败 → 输出原始格式 + 格式修改说明
