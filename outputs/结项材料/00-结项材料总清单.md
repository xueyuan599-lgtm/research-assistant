### 步骤2：清理临时文件

```bash
# 删除临时调试文件
find . -name "__pycache__" -type d -exec rm -rf {} +
find . -name "*.pyc" -delete
find . -name ".DS_Store" -delete
```

## 三、功能测试报告编写指南

请为以下核心模块编写测试用例并运行验证：

### 测试项清单

| 模块 | 测试内容 | 验证方法 |
|------|---------|---------|
| Orchestrator | 意图识别、路由调度 | 输入3种不同类型的任务，检查是否正确路由 |
| Secretary | 任务分解、领域判断 | 输入文献/数据/问答类任务，检查分解方案质量 |
| Literature Agent | 文献搜索、筛选、综述 | 输入检索词，检查搜索结果数量和综述结构 |
| Data Viz Agent | 数据清洗、建模、作图 | 用测试数据集运行全流程，检查产出文件 |
| Experiment Agent | 参数优化、敏感性分析 | 输入测试函数，检查优化结果 |
| Paper Format Agent | 格式适配、参考文献格式化 | 用测试文档运行格式检查 |
| Algorithm Pipeline | 形式化→设计→编码→验证 | 用简单算法想法测试全流水线 |
