## 一、系统概述

**Research Assistant** 是一款面向科学研究全过程的智能辅助系统，基于大语言模型驱动的多智能体（Multi-Agent）架构，将科研流程中的文献调研、方法选择、实验设计、数据分析、论文撰写等环节整合为统一平台，实现"一个需求输入，全流程智能响应"的科研辅助体验。

### 核心定位
- **输入**：自然语言描述的科研需求（如"帮我查2024年DID方法的前沿进展"）
- **输出**：可直接使用的科研成果（文献综述、分析报告、代码、图表、论文）

## 二、功能模块详情

### 模块1：科研知识问答（Research QA Agent）

**功能描述：** 回答科研方法原理、统计概念、公式推导等专业问题。

| 子功能 | 说明 |
|--------|------|
| 方法解释 | 解释经济学、统计学、机器学习方法的原理和适用条件 |
| 公式推导 | 从基础假设开始推导关键公式，展示推导过程 |
| 代码示例 | 为方法提供可直接运行的代码实现示例 |

**典型应用：** "解释DID方法中平行趋势假设的含义和检验方法"

### 模块2：文献智能检索（Literature Agent）

**功能描述：** 学术文献搜索、筛选、分析和自动综述。

| 子功能 | 说明 |
|--------|------|
| 学术搜索 | 集成OpenAlex（2.5亿+学术作品）、ArXiv API |
| 文献筛选 | 基于主题相关性和引用影响力的智能筛选 |
| 自动综述 | 生成结构化文献综述，含主题分类和方法论对比 |
| Meta分析辅助 | 提取文献中的效应量和统计量 |

**典型应用：** "帮我检索2023-2024年关于 staggered DiD 方法的文献综述"

### 模块3：论文选题分析（Topic Analysis Agent）

**功能描述：** 基于文献计量方法的研究前沿探测和选题推荐。

| 子功能 | 说明 |
|--------|------|
| 前沿探测 | 共词分析、引文聚类识别研究热点 |
| 研究空白识别 | 基于主题模型的文献覆盖度分析 |
| 趋势分析 | 研究主题的时间演化追踪 |
| 选题推荐 | 结合前沿和空白的创新选题建议 |

**典型应用：** "分析因果推断领域近3年的研究热点和未来方向"

### 模块4：科研数据处理与可视化（Data Viz Agent）

**功能描述：** 从原始数据到出版级图表的全流程自动化。

| 子功能 | 说明 |
|--------|------|
| 数据清洗 | 缺失值处理、异常值修正、类型校正、重复值剔除 |
| 统计建模 | 回归分析、分类、聚类、降维、时间序列分析 |
| 出版级图表 | 专业配色、期刊规范适配、高分辨率输出 |
| 结果解读 | 自动生成数据解读文本 |

**支持的语言工具：**
| 任务 | 首选 | 次选 |
|------|------|------|
| 统计建模 | R (lm, lme4) | Python (statsmodels) |
| 机器学习 | Python (scikit-learn) | R (caret) |
| 可视化 | Python (matplotlib, seaborn) | R (ggplot2) |
| 科学计算 | MATLAB | Python (SciPy) |

**典型应用：** "用这份面板数据做DID分析并生成事件研究图"

### 模块5：实验流程优化（Experiment Agent）

**功能描述：** 实验方案自动设计、参数优化和敏感性分析。

| 子功能 | 说明 |
|--------|------|
| 实验设计 | 根据研究问题自动设计对比实验方案 |
| 参数优化 | 网格搜索、贝叶斯优化、遗传算法 |
| 蒙特卡洛模拟 | 统计力分析、置信区间估计 |
| 敏感性分析 | 参数扰动、安慰剂检验、替代规格检验 |

**典型应用：** "帮我设计这个RDD方法的带宽选择敏感性分析"

### 模块6：论文格式与排版优化（Paper Format Agent）

**功能描述：** 期刊格式自动适配和论文排版辅助。

| 子功能 | 说明 |
|--------|------|
| 模板适配 | 支持各主流期刊模板（AER, JPE, Econometrica 等） |
| 参考文献管理 | 自动格式化、查重、补充 |
| 合规检查 | 检查格式、引用、长度等是否符合投稿要求 |
| LaTeX辅助 | LaTeX 模板配置、编译错误排查 |

**典型应用：** "将这份文档格式化为《经济研究》的投稿格式"

### 模块7：算法创造与知识沉淀（Algorithm + Knowledge Agent）

**功能描述：** 新算法从想法到实现的完整流水线。

| 子功能 | 说明 |
|--------|------|
| 问题形式化 | 自然语言想法 → 数学问题定义 |
| 算法设计 | 伪代码、数学性质推导、复杂度分析 |
| 代码实现 | Python/R/MATLAB 代码 + 单元测试 |
| 基准对比 | 与既有方法的数值比较 |
| 知识入库 | 沉淀到算法知识库供复用 |

**算法知识库覆盖：** 60+个标准算法，涵盖因果推断、时间序列、机器学习、深度学习、优化、科学计算、生物信息等领域。

## 六、学术诚信与合规说明

1. **数据安全**：所有数据在本地处理，不上传至外部服务器
2. **原创性**：核心功能（Agent编排、工作流设计、代码节点）均为团队自主实现
3. **可审查性**：完整源代码、开发日志、测试记录均可供审查
4. **引用合规**：API调用遵循OpenAlex、ArXiv等数据库使用条款
5. **不违规采集**：不使用未经授权的学生/教师个人信息
